# 计划书  
## 身份系统  

### 1. 神兽  
- 永久buff：抗性提升2、抗火、生命提升3、力量2、速度2、跳跃提升2  
- 仅腐竹可设置（青龙/白虎/朱雀/玄武）  
- 死亡不重置身份

---  
### 2. 生肖  

#### a. 人级  
- 佩戴对应`生肖面具`（头部装饰，addon物品）即可获得身份  摘下面具死亡
- 死亡后恢复为`参与者`  
- 禁用技能  

#### b. 地级  
- 条件：人级生肖积累3600点`道`（记分板）  
- 晋升时扣除3600道，弹出公告提示，头部装饰变为`生肖头颅`  摘下头颅死亡
- 永久增益：力量2、生命提升2  
- 地级生肖杀死玩家数量需要记录
- 死亡后恢复为`参与者`  清空杀人数量
- 禁用技能

#### c. 天级  
- 没有头部装饰
- 永久增益：力量2、生命提升2  
- 死亡后恢复为`参与者`  

---  
### 参与者  
- 默认玩家身份  

---  
## 核心机制  

### 1. 身份指令  
- `setrole set 选择器 身份ID` → 设置身份（管理员专用）  
- `setrole list` → 查看身份ID对照表（共41种身份数值）  

### 2. 生肖菜单  
功能：  
- 当前生肖类型/晋升进度  （道，杀人数）
- 游戏场地坐标/名称  
- 传送按钮（列车/游戏场地）  

### 3. 生肖游戏场地  
- **人级**：需划定50×50×50领地并命名，输入介绍  
- **地级**：需重新划定70×70×70领地并命名，输入介绍  
- **朱雀/天羊**额外拥有管理权限（可修改所有场地）  
- 玩家进入领地弹出弹出公告，内容为领地介绍

### 4. 列车房间  
- 固定坐标，生肖可通过菜单传送  

### 5. 死亡掉落  
- 对应回响者死亡时生成对应`眼球`道具（无回响不生成）

#回响系统

## 通用规则
- 开始和结束使用统一逻辑
- 结束事件必须清理所有数值、状态和tag
- 信念使用记分板`信念`记录
- `循环释放`回响需手动结束，`单次释放`回响释放后自动结束
- 统一释放结束事件跟随endll中的
